## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI arg -> path resolution | --world-dir accepts user-provided filesystem path |
| env var -> path resolution | CHRONICLE_WORLD_DIR accepts external string |
| world-manifest.json -> runtime | External JSON parsed and consumed at startup |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-01 | Tampering | --world-dir path traversal | mitigate | guardWorldDirPath() rejects paths with '..' segments after normalization; called before any fs access |
| T-10-02 | Tampering | world-manifest.json | mitigate | Zod schema validation via WorldManifestSchema.safeParse() before consuming any manifest field |
| T-10-03 | Information Disclosure | resolveDataDir error messages | mitigate | Error messages include only the failing path, not system state or env var values |
</threat_model>
