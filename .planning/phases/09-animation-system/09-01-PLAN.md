    - Test: calling trigger() sets active to true
    - Test: after durationMs elapses, active reverts to false
    - Test: calling trigger() while already active resets the timer (does not stack)
    - Test: cleanup clears timer on unmount (no setState after unmount)

    useTypewriter:
    - Test: displayText starts as empty string
    - Test: displayText accumulates characters at charIntervalMs rate
    - Test: isComplete becomes true when all characters are revealed
    - Test: calling skip() immediately sets displayText to full text and isComplete to true
    - Test: skip() while already complete is a no-op
  </behavior>
  <action>
    **useTimedEffect** (`src/ui/hooks/use-timed-effect.ts`):
    - Import `{ useState, useCallback, useRef, useEffect }` from 'react'
    - Export type `UseTimedEffectReturn = { readonly active: boolean; readonly trigger: () => void }`
    - Export function `useTimedEffect(durationMs: number): UseTimedEffectReturn`
    - Implementation: `useState(false)` for active, `useRef<ReturnType<typeof setTimeout> | null>(null)` for timerRef
    - `trigger`: clearTimeout if timerRef.current, setActive(true), setTimeout to setActive(false) after durationMs, store in timerRef
    - `useEffect` cleanup: clearTimeout(timerRef.current) on unmount
    - Wrap trigger in `useCallback([durationMs])`

    **useTypewriter** (`src/ui/hooks/use-typewriter.ts`):
    - Import `{ useState, useCallback, useEffect, useRef }` from 'react'
    - Export type `UseTypewriterReturn = { readonly displayText: string; readonly isComplete: boolean; readonly skip: () => void }`
    - Export function `useTypewriter(fullText: string, charIntervalMs: number): UseTypewriterReturn`
    - Implementation: `useState(0)` for charCount, `useRef(fullText)` for fullTextRef (kept in sync)
    - `isComplete = charCount >= fullText.length`
    - `useEffect`: if not complete, setInterval incrementing charCount. clearInterval when charCount reaches fullText.length. Return cleanup.
    - `skip`: `setCharCount(fullText.length)` via useCallback([fullText.length])
    - `displayText = fullText.slice(0, charCount)`

    **Tests**: Use `bun:test` with `describe/it/expect`. Test hook logic by calling the functions directly and checking return values. For timer-dependent tests, use `import { jest } from 'bun:test'` with `jest.useFakeTimers()` and `jest.advanceTimersByTime()`. If bun:test does not support fake timers, test the logic by verifying function signatures and initial return values, plus manual timer tests with real timers and short durations (50ms).
  </action>
