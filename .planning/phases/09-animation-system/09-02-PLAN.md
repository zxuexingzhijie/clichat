    Rewrite `title-screen.tsx` to add animated typewriter reveal. Keep `generateTitleArt()` helper. Key changes:

    **Gradient pre-computation (per Research Pitfall #5 and D-01):**
    The main challenge is that `gradient-string.multiline()` produces ANSI escape sequences that cannot be sliced. Instead, pre-compute per-character colors:
    1. Generate figlet text with `figlet.textSync('CHRONICLE', { font: 'ANSI Shadow' })`
    2. Split into rows. Find max column width.
    3. For each column index 0..maxCols, compute a gradient color by linear interpolation between cyan (#00FFFF) and magenta (#FF00FF) based on `columnIndex / maxCols`. Use `chalk.hex(hexColor)` to colorize each character.
    4. Store the result as an array of colored rows (string[]) — the full pre-colored art.
    5. Also store a plain text version (rows without ANSI) for character counting.

    Import chalk: `import chalk from 'chalk'`

    **Column-reveal typewriter (per D-01, D-02, Research Pitfall #4):**
    The typewriter must reveal column-by-column across the multi-line figlet block, not character-by-character on a flat string. This is because figlet characters span multiple rows.

    1. Compute `totalColumns` = max row length of the figlet output
    2. Use `useTypewriter` with `fullText` set to a dummy string of length `totalColumns` (e.g., `' '.repeat(totalColumns)`). This drives the column counter at ~25ms per column.
    3. `displayText.length` gives the number of visible columns.
    4. For rendering: for each pre-colored row, take the first `visibleCols` characters and pad the rest with spaces. This reveals columns left-to-right.

    ANSI-aware slicing: Since we pre-compute colors per-character (not using `gradient-string.multiline`), each character is a separate chalk call. The rows array already has individually colored characters. Slicing by column is safe because each character is an independent ANSI segment.

    Implementation of per-character gradient coloring:
    ```typescript
    function interpolateHex(startHex: string, endHex: string, t: number): string {
      const r1 = parseInt(startHex.slice(1, 3), 16);
      const g1 = parseInt(startHex.slice(3, 5), 16);
      const b1 = parseInt(startHex.slice(5, 7), 16);
      const r2 = parseInt(endHex.slice(1, 3), 16);
      const g2 = parseInt(endHex.slice(3, 5), 16);
      const b2 = parseInt(endHex.slice(5, 7), 16);
      const r = Math.round(r1 + (r2 - r1) * t);
      const g = Math.round(g1 + (g2 - g1) * t);
      const b = Math.round(b1 + (b2 - b1) * t);
      return '#' + [r, g, b].map(v => v.toString(16).padStart(2, '0')).join('');
    }

    function colorizeArt(art: string): { coloredRows: string[]; totalColumns: number } {
      const rows = art.split('\n');
      const maxLen = Math.max(...rows.map(r => r.length));
      const coloredRows = rows.map(row => {
        let coloredLine = '';
        for (let col = 0; col < row.length; col++) {
          const char = row[col]!;
          if (char === ' ') {
            coloredLine += ' ';
          } else {
            const t = maxLen > 1 ? col / (maxLen - 1) : 0;
            const hex = interpolateHex('#00FFFF', '#FF00FF', t);
            coloredLine += chalk.hex(hex)(char);
          }
        }
        return coloredLine;
      });
      return { coloredRows, totalColumns: maxLen };
    }
    ```

    ```typescript
    function colorizeArt(art: string): { coloredGrid: string[][]; totalColumns: number } {
      const rows = art.split('\n');
      const maxLen = Math.max(...rows.map(r => r.length));
      const coloredGrid = rows.map(row => {
        const chars: string[] = [];
        for (let col = 0; col < maxLen; col++) {
          const char = col < row.length ? row[col]! : ' ';
          if (char === ' ') {
            chars.push(' ');
          } else {
            const t = maxLen > 1 ? col / (maxLen - 1) : 0;
            const hex = interpolateHex('#00FFFF', '#FF00FF', t);
            chars.push(chalk.hex(hex)(char));
          }
        }
        return chars;
      });
      return { coloredGrid, totalColumns: maxLen };
    }
    ```
