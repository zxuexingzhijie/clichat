    Types:
    ```typescript
    export type NpcDialogueContext = {
      readonly npcProfile: NpcProfile;
      readonly scene: string;
      readonly playerAction: string;
      readonly memories: readonly string[];
    };

    export type UseNpcDialogueReturn = {
      readonly streamingText: string;
      readonly isStreaming: boolean;
      readonly error: Error | null;
      readonly metadata: NpcDialogue | null;
      readonly startDialogue: (context: NpcDialogueContext) => void;
      readonly skipToEnd: () => void;
      readonly reset: () => void;
    };
    ```

    Hook implementation `useNpcDialogue(): UseNpcDialogueReturn`:

    State:
    - `streamingText` (useState<string>(''))
    - `isStreaming` (useState<boolean>(false))
    - `error` (useState<Error | null>(null))
    - `metadata` (useState<NpcDialogue | null>(null))

    Refs:
    - `cancelledRef` (useRef<boolean>(false))
    - `skippedRef` (useRef<boolean>(false))
    - `fullTextRef` (useRef<string>(''))
    - `bufferRef` (useRef<SentenceBuffer | null>(null))
    - `contextRef` (useRef<NpcDialogueContext | null>(null)) -- store context for fallback

    `startDialogue(context)`:
    1. Reset refs and state. Store context in contextRef.
    2. Create sentence buffer with `onFlush: (text) => setStreamingText(prev => prev + text)`
    3. Emit `eventBus.emit('npc_dialogue_streaming_started', { npcId: context.npcProfile.id, npcName: context.npcProfile.name })`
    4. Start async IIFE:
       ```
       for await (const chunk of streamNpcDialogue(npcProfile, scene, playerAction, memories)):
         if (cancelledRef.current) break;
         fullTextRef.current += chunk;
         if (!skippedRef.current):
           bufferRef.current?.push(chunk);
       ```
    5. After stream completes:
       - Flush and dispose buffer
       - `setStreamingText(fullTextRef.current)`
       - Run metadata extraction:
         ```
         const extracted = extractNpcMetadata(fullTextRef.current);
         const isAllDefaults = extracted.emotionTag === 'neutral' && !extracted.shouldRemember && extracted.relationshipDelta === 0;
         const isSubstantive = fullTextRef.current.length > 50;
         ```
       - If `isAllDefaults && isSubstantive` (per D-06 / Pitfall 5: regex returned all defaults on substantive dialogue), fall back to generateNpcDialogue:
         ```
         try {
           const fallbackResult = await generateNpcDialogue(npcProfile, scene, playerAction, memories);
           setMetadata({ ...fallbackResult, dialogue: fullTextRef.current });
         } catch {
           setMetadata({ dialogue: fullTextRef.current, ...extracted });
         }
         ```
       - Otherwise: `setMetadata({ dialogue: fullTextRef.current, ...extracted })`
       - Emit `eventBus.emit('npc_dialogue_streaming_completed', { npcId: npcProfile.id, charCount: fullTextRef.current.length })`
       - `setIsStreaming(false)`
    6. On catch: `setError(...)`, `setIsStreaming(false)`

    `skipToEnd`: Same pattern as useAiNarration -- set skippedRef, flush/dispose buffer, setStreamingText(fullTextRef.current)

    `reset`: Cancel, dispose buffer, clear all state.

    Return: `{ streamingText, isStreaming, error, metadata, startDialogue, skipToEnd, reset }`
