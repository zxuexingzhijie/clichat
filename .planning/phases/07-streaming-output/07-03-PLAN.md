    ```typescript
    import { createSentenceBuffer } from '../../ai/utils/sentence-buffer';
    import { eventBus } from '../../events/event-bus';
    ```

    Internal state:
    - `streamingText` (useState<string>('')): text flushed from buffer so far (displayed in ScenePanel as streaming line)
    - `isStreaming` (useState<boolean>(false))
    - `error` (useState<Error | null>(null))
    - `cancelledRef` (useRef<boolean>(false)): existing cancel mechanism
    - `skippedRef` (useRef<boolean>(false)): NEW -- controls whether buffer is bypassed
    - `fullTextRef` (useRef<string>('')): NEW -- accumulates ALL received text regardless of skip state
    - `bufferRef` (useRef<SentenceBuffer | null>(null)): NEW -- holds active buffer instance

    `startNarration` implementation:
    1. Reset all refs and state: `cancelledRef.current = false`, `skippedRef.current = false`, `fullTextRef.current = ''`, `setStreamingText('')`, `setIsStreaming(true)`, `setError(null)`
    2. Create sentence buffer: `bufferRef.current = createSentenceBuffer({ onFlush: (text) => { setStreamingText(prev => prev + text); } })`
    3. Emit: `eventBus.emit('narration_streaming_started', { sceneType: context.sceneType })`
    4. Start async IIFE:
       ```
       for await (const chunk of streamNarration(context)):
         if (cancelledRef.current) break;
         fullTextRef.current += chunk;
         if (!skippedRef.current):
           bufferRef.current?.push(chunk);
       ```
    5. After loop completes (in finally or after loop):
       - `bufferRef.current?.flush()` then `bufferRef.current?.dispose()`
       - `setStreamingText(fullTextRef.current)` (ensure final text is complete -- silently replace per D-08)
       - `setIsStreaming(false)`
       - `eventBus.emit('narration_streaming_completed', { charCount: fullTextRef.current.length })`
    6. On catch (if not cancelled): `setError(err)`, `setIsStreaming(false)`

    `skipToEnd` implementation (per D-08):
    ```typescript
    const skipToEnd = useCallback(() => {
      if (!skippedRef.current && isStreaming) {
        skippedRef.current = true;
        bufferRef.current?.flush();
        bufferRef.current?.dispose();
        setStreamingText(fullTextRef.current);
      }
    }, [isStreaming]);
    ```

    `reset` implementation (keep existing pattern):
    ```typescript
    const reset = useCallback(() => {
      cancelledRef.current = true;
      bufferRef.current?.dispose();
      setStreamingText('');
      setIsStreaming(false);
      setError(null);
    }, []);
    ```

    Return: `{ streamingText, isStreaming, error, startNarration, skipToEnd, reset }`
