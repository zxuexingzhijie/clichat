## 角色创建流程

| Option | Description | Selected |
|--------|-------------|----------|
| 向导式步骤流程 | 分步走：选种族→选职业→调整属性→确认。每步有解释文本 | |
| 单屏快速选择 | 一屏展示所有选项，快速确认开始游戏 | |
| 混合模式 | 第一次玩向导式，之后可切换快速模式 | ✓ |

**User's choice:** 混合模式 — 首次向导式剧情流程 + 后续快速模式（预设/随机/自定义）
**Notes:** 用户强调"角色创建本身就应该是第一段剧情"。向导问题用叙事包装（"你从哪里来？"），不是冷冰冰的表格。背景钩子影响初始世界状态。

### 属性分配方式

| Option | Description | Selected |
|--------|-------------|----------|
| 纯预设（选择即决定） | 种族/职业/背景组合直接决定三属性值和标签 | |
| 微调式（少量自由点） | 种族/职业给基础值，玩家有2-3点自由分配 | |
| 叙事决定（剧情选择暗含属性） | 向导过程中通过剧情选择隐含决定属性偏向 | ✓ |

**User's choice:** 叙事决定 — 玩家在向导中的选择暗含属性倾向，不直接看到数字

## AI叙述风格

| Option | Description | Selected |
|--------|-------------|----------|
| 古风文言 | 类似古小说/武侠小说口吴 | |
| 轻小说风 | 类似网络小说/轻小说口吴，轻快易读 | |
| 电影感白话 | 类似电影剧本描写，简洁画面感 | |
| 动态混合风格 | 根据场景调性动态切换 | ✓ |

**User's choice:** 动态混合风格 — 按场景类型切换语言节奏
**Notes:** 探索=电影感白话，战斗=短句强动作，对话=轻小说自然口语，历史传说=轻度古风，恐怖/神秘=压低信息量，检定结果=清楚解释再接叙事

### 叙述视角

| Option | Description | Selected |
|--------|-------------|----------|
| 第二人称（你） | "你拔出剑" — 沉浸感强 | |
| 第三人称 | "少年拔出剑" — 更像小说 | |
| 混合（场景决定） | 实时用"你"，传说用第三人称 | ✓ |

**User's choice:** 混合视角 — 实时/战斗/探索第二人称，历史/回顾第三人称，系统日志客观陈述

## NPC对话系统

| Option | Description | Selected |
|--------|-------------|----------|
| 内联流式（无单独对话框） | 对话嵌在场景面板中 | |
| 独立对话模式（场景切换） | 进入对话时切换专门布局 | |
| 混合（重要度决定） | 普通内联，关键对话自动升级 | ✓ |

**User's choice:** 混合呈现 — 普通短对话内联，关键对话自动进入Dialogue Mode
**Notes:** Dialogue Mode在Scene面板中切换布局，显示NPC名称、台词、关系状态、可选回应

### NPC情绪可见性

| Option | Description | Selected |
|--------|-------------|----------|
| 可见情绪标签 | 台词前显示如「警惕」「友善」 | |
| 纯叙事体现（无标签） | 仅通过台词语气和行为描写体现 | |
| 检定解锁（心智属性关联） | 心智检定成功后才显示情绪标签 | ✓ |

**User's choice:** 叙事默认 + 检定解锁 — 默认通过描写体现，心智检定/背景/关系可解锁明确提示
