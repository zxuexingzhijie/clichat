**Create `src/engine/combat-loop.ts`:**
Factory: `createCombatLoop(codexEntries, options?: { rng?, generateNarrationFn? })`
Returns: `{ startCombat, processPlayerAction, processEnemyTurn, checkCombatEnd, getCombatPhase }`

`startCombat(enemyIds: string[])`:
1. Look up enemy data from codex (EnemySchema entries)
2. Build enemies array: `{ id, name, hp: enemy.maxHp, maxHp: enemy.maxHp }`
3. Build turn order: `['player', ...enemyIds]`
4. Update combatStore: active=true, phase='player_turn', enemies, turnOrder, roundNumber=1
5. Set gameStore phase to 'combat'
6. Generate opening narration with sceneType 'combat'
7. Update combatStore.lastNarration

`processPlayerAction(actionType: 'attack' | 'cast' | 'guard' | 'flee', targetIndex?: number)`:
1. Set combatStore phase to 'resolving'
2. Get player state, get target enemy (default: first alive enemy)
3. Determine check parameters:
   - attack: attribute='physique', dc=enemy.dc, weaponBase from player equipment (codex item lookup, default 5)
   - cast: attribute='mind', dc=enemy.dc, check player MP >= 4. If insufficient: return error "魔力不足！". Deduct 4 MP.
   - guard: no check needed. Set combatStore.guardActive=true. Skip to narration.
   - flee: attribute='finesse', dc=10
4. Roll + adjudicate via `resolveNormalCheck({ roll: rollD20(rng), attributeName, attributeModifier, skillModifier: 0, environmentModifier: 0, dc })`
5. Store checkResult in combatStore.lastCheckResult
6. If attack/cast success: `calculateDamage({ weaponBase, attributeModifier: attrMod, grade, armorReduction: enemy.defense })`
7. Apply damage: update enemy HP in combatStore
8. If flee success: end combat with outcome 'flee'
9. Set phase to 'narrating'
10. Generate narration via `generateNarrationFn` with sceneType 'combat', passing checkResult
11. Store narration in combatStore.lastNarration
12. Check combat end. If not ended: advance to 'enemy_turn'

`processEnemyTurn()`:
1. For each alive enemy (in turn order):
2. Set phase to 'resolving'
3. Enemy rolls: `resolveNormalCheck({ roll: rollD20(rng), attributeName: 'physique', attributeModifier: enemy.attack, skillModifier: 0, environmentModifier: 0, dc: playerAC })` where playerAC = 10 + player armor_value + (guardActive ? 2 : 0)
4. If success: `calculateDamage({ weaponBase: enemy.damage_base, ... })`, reduce player HP
5. Reset guardActive to false
6. Store check result and generate narration
7. After all enemies acted: advance to 'check_end' then 'player_turn' if combat continues

`checkCombatEnd()`:
- All enemies HP <= 0: end combat, outcome 'victory', narration "战斗胜利！{enemy}被击败了。", return to game phase
- Player HP <= 0: end combat, outcome 'defeat', narration "你倒下了...... 世界在眼前逐渐模糊。"
- Otherwise: continue combat
