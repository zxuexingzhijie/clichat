Create `src/engine/scene-manager.ts`:
- Factory pattern: `createSceneManager(codexEntries: Map<string, CodexEntry>, options?: { generateNarrationFn?, generateRetrievalPlanFn? })`
- Dependency injection for AI functions (enables mocking in tests)
- Returns interface: `loadScene(locationId)`, `handleLook(target?)`, `handleInspect(target)`, `handleGo(direction)`, `getCurrentScene()`

`loadScene(locationId: string)`:
1. Get location from codex via `queryById(codexEntries, locationId)`
2. If not found, return error
3. Update `sceneStore.setState()` with: sceneId=locationId, locationName=location.name, npcsPresent=location.notable_npcs, exits=location.exits, objects=location.objects
4. Generate retrieval plan via injected `generateRetrievalPlanFn` (or imported `generateRetrievalPlan`)
5. Assemble context via `assembleNarrativeContext`
6. Generate narration via injected `generateNarrationFn` (or imported `generateNarration`) with sceneType 'exploration'
7. Update `sceneStore.setState()` with narrationLines
8. Build suggested actions from location data:
   - For each NPC: `{ id: 'talk_{npcId}', label: '与{npcName}交谈', type: 'talk' }`
   - For each object: `{ id: 'inspect_{objId}', label: '检查{objName}', type: 'inspect' }`
   - For each exit: `{ id: 'go_{direction}', label: '前往{direction}', type: 'move' }`
9. Update `sceneStore.setState()` with actions
10. Emit `scene_changed` event via eventBus

`handleLook(target?: string)`:
- No target: return current `sceneStore.getState().narrationLines`
- With target: find target in npcsPresent or objects, generate AI narration with sceneType 'exploration' focused on that target

`handleInspect(target: string)`:
- Find target in codex (try as NPC ID, then as item/object ID)
- Generate detailed AI narration about the target
- Append to narrationLines

`handleGo(direction: string)`:
- Check if direction is in current scene's exits
- If not: return `{ status: 'error', message: '那个方向没有出路。' }`
- If yes: find the location connected to that exit (convention: exit string is the location_id or a direction->location_id map)
- Call `loadScene(targetLocationId)`

- Test: loadScene updates sceneStore with location data
- Test: handleLook returns narration lines
