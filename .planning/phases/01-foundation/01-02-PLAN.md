SuccessGrade enum values (from D-16 + Claude discretion):
  'critical_success' | 'great_success' | 'success' | 'partial_success' | 'failure' | 'critical_failure'

Grade thresholds:
  - nat20 -> critical_success (always, regardless of modifiers)
  - nat1 -> critical_failure (always, regardless of modifiers)
  - total >= DC + 10 -> great_success
  - total >= DC -> success
  - total >= DC - 5 -> partial_success
  - total < DC - 5 -> failure

D-19 Damage formula:
  total = weaponBase + attributeModifier + gradeBonus - armorReduction
  total = Math.max(0, total)  // no negative damage

D-20 Check display format:
  `[D20: ${roll}] + ${attrLabel} ${attrMod} = ${total} vs DC ${dc} → ${gradeLabel}`

  <behavior>
    Dice module:
    - rollD20() returns integer 1-20
    - rollD20(seed) with same seed returns same result (deterministic tests)
    - rollPercentage() returns integer 1-100
    - createSeededRng(seed) returns a function that produces repeatable sequences

    Normal check (D-13):
    - Given roll=14, attrMod=3, skillMod=0, envMod=0, dc=15: total=17, grade=success
    - Given roll=20 (nat20), any modifiers, any DC: grade=critical_success
    - Given roll=1 (nat1), any modifiers, any DC: grade=critical_failure
    - Given roll=15, attrMod=3, skillMod=2, envMod=0, dc=10: total=20, grade=great_success (>= DC+10)
    - Given roll=8, attrMod=2, skillMod=0, envMod=0, dc=15: total=10, grade=failure (< DC-5)
    - Given roll=10, attrMod=2, skillMod=0, envMod=0, dc=15: total=12, grade=partial_success (>= DC-5, < DC)

    Opposed check (D-14):
    - Both sides roll D20 + attribute + skill, higher total wins
    - Tie goes to defender (the second participant)
    - Returns CheckResult for each side plus winner

    Probability check (D-15):
    - Roll percentage (1-100) vs threshold
    - Roll <= threshold: success. Roll > threshold: failure.

    Plot-critical check (D-16):
    - Same as normal check but all 6 grades are used for narrative branching
    - Returns grade + narrative_hint field suggesting AI narration tone

    Damage (D-19):
    - weaponBase=5, attrMod=3, gradeBonus=2, armor=3 -> total=7
    - weaponBase=5, attrMod=1, gradeBonus=0, armor=8 -> total=0 (floor at 0)
    - Grade bonus mapping: critical_success=4, great_success=3, success=2, partial_success=1, failure=0, critical_failure=0

    Display (D-20):
    - Normal check display: "[D20: 14] + 体魄 3 = 17 vs DC 15 → 成功！"
    - Critical display: "[D20: 20] → 大成功！"
  </behavior>
  <implementation>
Create `src/engine/types.ts` with CheckParams, OpposedCheckParams, ProbabilityCheckParams, DamageParams interfaces.

Create `src/engine/dice.ts`:
- `createSeededRng(seed: number)`: Simple seedable PRNG (mulberry32 algorithm — 4 lines of math, well-known, no deps needed). Returns `() => number` that produces floats 0-1 deterministically.
- `rollD20(rng?: () => number)`: Uses provided RNG or Math.random. Returns Math.floor(rng() * 20) + 1.
- `rollPercentage(rng?: () => number)`: Returns Math.floor(rng() * 100) + 1.

Create `src/engine/adjudication.ts`:
- `resolveNormalCheck(params: CheckParams, rng?)`: D-13 logic. Computes total, determines grade, builds display string.
- `resolveOpposedCheck(params: OpposedCheckParams, rng?)`: D-14 logic. Two rolls, compare totals. Tie -> defender wins.
- `resolveProbabilityCheck(params: ProbabilityCheckParams, rng?)`: D-15 logic. Percentage vs threshold.
- `resolvePlotCriticalCheck(params: CheckParams, rng?)`: D-16 logic. Same grading as normal but adds narrative_hint field.

Create `src/engine/damage.ts`:
- `calculateDamage(params: DamageParams)`: D-19 formula. Floor at 0. Grade bonus lookup table.
- `getGradeBonus(grade: SuccessGrade)`: Returns bonus number per grade.

Create `src/engine/rules-engine.ts`:
- `resolveAction(action, context, rng?)`: Facade that routes to appropriate adjudication mode based on action type and context. Returns unified result.

Chinese attribute labels for display: physique='体魄', finesse='技巧', mind='心智'.
Check display format per D-20: `[D20: ${roll}] + ${attrLabel} ${attrMod} = ${total} vs DC ${dc} → ${gradeLabel}`
Critical display shortcut: `[D20: 20] → 大成功！` / `[D20: 1] → 大失败！`
