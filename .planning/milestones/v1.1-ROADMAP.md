## Milestone Summary

**Key Decisions:**
- SIGINT handler uses try/catch with process.exit(0) fallback for startup race
- BUG-02 branches placed BEFORE overlay-panel escape check
- Weight resolver uses standalone pure functions with 4-layer deterministic tiebreaker
- Pre-compute per-character gradient with chalk.hex() instead of gradient-string.multiline()
- Flash scope limited to HP value text only; Nat20/Nat1 use inverse flash
- Spinner dimout uses 3-state machine for smooth transition
- guardWorldDirPath uses segment splitting (not path.normalize) to detect '..'
- CLI sets process.env.__CHRONICLE_DATA_DIR before dynamic import
- Workflow uses env: blocks instead of inline GitHub expression interpolation to mitigate sed injection
