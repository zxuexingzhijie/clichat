    **GREEN phase -- create weight-resolver.ts:**

    Export these types:
    ```typescript
    export type DialogueOptionEffect = {
      readonly raceId?: string;
      readonly professionWeights: Readonly<Record<string, number>>;
      readonly backgroundWeights: Readonly<Record<string, number>>;
      readonly tags: readonly string[];
    };

    export type AccumulatedWeights = {
      readonly raceId: string;
      readonly professionWeights: Readonly<Record<string, number>>;
      readonly backgroundWeights: Readonly<Record<string, number>>;
      readonly tags: readonly string[];
      readonly roundEffects: readonly DialogueOptionEffect[];
    };

    export type TiebreakerConfig = {
      readonly archetypePriority: {
        readonly profession: readonly string[];
        readonly background: readonly string[];
      };
      readonly questionPriority: {
        readonly profession: number;  // which round (1-4) matters most for profession
        readonly background: number;  // which round (1-4) matters most for background
      };
    };
    ```

    Export these pure functions:

    `createInitialWeights()`: Returns empty AccumulatedWeights with raceId='', empty records, empty tags, empty roundEffects.

    `accumulateWeights(current: AccumulatedWeights, effect: DialogueOptionEffect, roundIndex: number): AccumulatedWeights`:
    - Returns a NEW object (spread + Object.entries reduce for weight merging)
    - Sets raceId from effect.raceId if present (round 1), otherwise preserves current.raceId
    - Sums professionWeights: for each key in effect.professionWeights, add to current value (default 0)
    - Same for backgroundWeights
    - Concatenates tags
    - Appends effect to roundEffects array

    `resolveCharacter(weights: AccumulatedWeights, config: TiebreakerConfig): { raceId: string; professionId: string; backgroundIds: readonly string[] }`:
    - professionId = resolveByWeight(weights.professionWeights, 'profession', weights, config)
    - backgroundIds = resolveTopN(weights.backgroundWeights, 2, 'background', weights, config)
    - Returns { raceId: weights.raceId, professionId, backgroundIds }
    - The returned object has `name: ''` (name is filled separately per D-05)

    Internal `resolveByWeight(weights, category, accumulated, config)`:
    1. Sort candidates by weight descending
    2. If clear winner (no tie), return it
    3. Tie? Layer 1: Among tied candidates, which has higher weight from the LAST round's effect? (roundEffects[3])
    4. Still tied? Layer 2: Compare weight contribution from the question-priority round (config.questionPriority[category])
    5. Still tied? Layer 3: archetypePriority[category] -- earlier index wins
    6. Still tied? Layer 4: First candidate in iteration order (stable sort preserves insertion order)

    Default TiebreakerConfig:
    ```typescript
    questionPriority: { profession: 1, background: 2 }
    // Round 2 (index 1) is "livelihood" -> profession priority
    // Round 3 (index 2) is "reason for visiting" -> background priority
    ```
