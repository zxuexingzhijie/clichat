Purpose: These utilities are the data-processing backbone of the streaming pipeline. The sentence buffer prevents per-character React re-renders (per D-01, D-02). The metadata extractor enables the stream-first, extract-later NPC dialogue pattern (per D-05).

Output: Two tested utility modules with zero external dependencies.
